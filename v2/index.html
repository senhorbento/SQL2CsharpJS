SQL to C# Entity

Criar CRUD

Como usar?

  1. Primeiro insira uma tabela em SQL na primeira área de texto
  2. Depois clique no botão Converter.
  3. O resultado será mostrado na segunda área de texto.
  4. Observação: É importante os atributos estarem um por linha como no exemplo abaixo.

Exemplo

Entrada:

CREATE TABLE PessoaFisica(

 id INT PRIMARY KEY AUTO_INCREMENT,

 nome VARCHAR(100) NOT NULL,

 idade SMALLINT NOT NULL,

 cpf VARCHAR(14) NOT NULL,

 telefone char(20) NOT NULL,

 data_nascimento DATE,

 saldo decimal

);

Saida:

public class PessoaFisica{

 public int id { get; set; }

 public string nome { get; set; }

 public short idade { get; set; }

 public string cpf { get; set; }

 public string telefone { get; set; }

 public DateTime data_nascimento { get; set; }

 public decimal saldo { get; set; }

}
